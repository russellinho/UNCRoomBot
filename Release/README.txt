How to install:
1. Install the provided Microsoft Visual C++ Redistributable 2015 located in the Requirements folder.
2. Install node.js provided in the Requirements folder.
3. Extract the chromedriver zip in the Requirements folder.
4. Make a new folder under C:\ path. Name it webdrivers. Move the chromedriver executable extracted in step 2 to here.
5. Program should be good to go. If you have issues, contact Russell Creswell.
